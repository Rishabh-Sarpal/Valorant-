# Valorant Agent Expert System

## Overview

This project implements a **rule-based expert system** that recommends the best Valorant agent based on a player's playstyle.
The system uses artificial intelligence concepts such as a **knowledge base, inference engine, and rule-based reasoning** to simulate decision-making like a human expert.

The program asks users questions about their preferred gameplay style (aggressive play, support, intelligence gathering, defense, mobility, etc.) and recommends the most suitable Valorant agents.

---

## Features

* Rule-based AI expert system
* Uses a knowledge base containing all Valorant agents
* Considers agent roles and abilities
* Provides agent recommendations based on playstyle
* Displays agent abilities and roles
* Score-based inference mechanism

---

## Technologies Used

* Python
* Google Colab
* Rule-Based Artificial Intelligence

---

## Expert System Components

### Knowledge Base

Contains information about:

* Valorant agents
* Agent roles (Duelist, Sentinel, Controller, Initiator)
* Agent abilities

### Inference Engine

Applies rules to determine the most suitable agent for the player.

Example rule:

IF player prefers aggressive gameplay
THEN increase score for Duelist agents

### User Interface

The system interacts with the user through questions to understand their playstyle.
